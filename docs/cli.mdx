---
title: CLI
description: Command-line interface for the Luma SDK.
---

After installing `@alhwyn/luma-cli`:

```bash
bun add @alhwyn/luma-cli
```

## Commands

```bash
luma users get
luma events list
luma --help
```

The CLI reads `LUMA_API_KEY` from your environment. See [Install](/install) for package setup.
