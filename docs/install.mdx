## SDK

From npm (recommended once published):

```bash
bun add @alhwyn/luma
```

If that returns a 404, install directly from GitHub (no token required):

```bash
bun add github:Alhwyn/luma
```

Both resolve to `@alhwyn/luma` in `node_modules` and use the same import path:

```ts
import { Luma } from "@alhwyn/luma";
```

## CLI (optional)

```bash
bun add @alhwyn/luma-cli
```

The CLI is published separately on npm. Use the SDK install above if you only need the TypeScript client.
