---
title: Install
description: Install @alhwyn/luma and @alhwyn/luma-cli from npm.
---

Packages are published on the public [npm registry](https://www.npmjs.com/package/@alhwyn/luma). No GitHub token or `.npmrc` setup is required.

## SDK

```bash
bun add @alhwyn/luma
```

## CLI (optional)

```bash
bun add @alhwyn/luma-cli
```

## Next steps

1. [Authentication](/authentication) — set up `LUMA_API_KEY`
2. [Test your setup](/test-your-setup) — verify against the live Luma API
3. [Usage](/usage) — start calling the SDK
